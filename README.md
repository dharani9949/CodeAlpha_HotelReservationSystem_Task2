## Overview of the Project

The Hotel Reservation System is a Java-based console application designed to manage hotel room bookings efficiently. It allows users to view available rooms, book rooms, cancel bookings, and view booking details using a simple menu-driven interface. This project demonstrates Object-Oriented Programming (OOP) concepts in Java.

---

## Objective

The objective of this project is to:

* Simulate a real-world hotel booking system.
* Practice Java OOP concepts.
* Work with arrays and classes.
* Handle user input using Scanner.
* Implement booking and cancellation logic.

---

## Features

* View available rooms with booking status.
* Book a room using room number.
* Cancel booked rooms.
* View booking details.
* Payment simulation.
* Menu-driven console interface.
