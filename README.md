📖 Overview of the Project

The Hotel Reservation System is a Java-based console application designed to manage hotel room bookings efficiently. It allows users to view available rooms, book a room, cancel bookings, and view booking details using a simple menu-driven interface. This project demonstrates Object-Oriented Programming (OOP) concepts in Java.

🎯 Objective
Simulate a real-world hotel booking system
Practice Java OOP concepts
Work with arrays and classes
Handle user input using Scanner
Implement booking and cancellation logic
⚙️ Features
View all available rooms with status
Book a room using room number
Cancel booked room
View booking details
Simple console-based menu system
🛠️ Technologies Used
Java
Object-Oriented Programming (OOP)
Arrays
Scanner Class
Loops and Switch Case
